# Accurate BMP2 for Gunner HEAT PC!

Version 1.0. of Accurate BMP2 mod. Makes the BMP2 main gun as accurate as the Bradley's.

***Installation***  
This mod requires MelonLoader Version 0.6.1.

To install, extract the dll file from releases and put it into the mod folder.
